FileGenie: Document Q&A System

Python, Flask, React, LangChain, FAISS, Groq, Google Generative AI

  • Engineered a powerful document Q&A system that allows users to upload PDF files and ask questions about their content, leveraging advanced language models for accurate responses.
  • Implemented a robust backend using Flask, integrating LangChain for efficient document processing and FAISS for high-performance vector similarity search.
  • Developed a user-friendly frontend with React, featuring an intuitive interface for file uploads and question submissions, enhancing the overall user experience.
  • Utilized Groq and Google Generative AI to process natural language queries and generate context-aware responses, providing users with precise information extracted from their documents.
  • Deployed the application on Render, ensuring scalability and accessibility for users to interact with their documents anytime, anywhere.
